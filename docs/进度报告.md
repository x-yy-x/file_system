## 已完成的模块

### 数据模型 (`com.fs.model`)

| 类 | 说明 |
|----|------|
| `SuperBlock` | 超级块，磁盘布局描述（固定 512B） |
| `Inode` | 索引节点（固定 64B，9直接块+一级间接+二级间接） |
| `DirectoryEntry` | 目录项（固定 48B，含 inode 号/文件名/类型） |
| `FileDescriptor` | 文件描述符（fd/偏移/模式） |
| `PathInfo` | 路径解析结果 |
| `EntryInfo` | ls 返回条目信息 |
| `FileStat` | stat 详细信息 |
| `SystemInfo` | 系统概要信息 |
| `FileSystemException` | 业务异常 |

### 磁盘驱动层 (`com.fs.driver`)

| 类 | 说明 |
|----|------|
| `DiskDriver` | 虚拟磁盘（16MB = 32768块 × 512B），盘块读写 + 文件持久化 |
| `Serializer` | SuperBlock / Inode / DirectoryEntry 序列化/反序列化 |

### 管理层 (`com.fs.manager`)

| 类 | 说明 |
|----|------|
| `BitmapManager` | 位图管理（4096字节 = 32768位），分配/回收 |
| `InodeManager` | i-node 管理（1024个），直接/间接块指针寻址 |
| `FileAllocator` | 文件数据读写，协调位图和 inode |
| `PathResolver` | 路径解析（绝对/相对路径，规范化，合法性校验） |
| `DirectoryManager` | 多级目录树（增删改查，树形显示，路径导航） |

### 核心 API + CLI

| 类 | 说明 |
|----|------|
| `FileSystem` | 统一操作入口：格式化/目录/文件/附加/持久化 |
| `CommandParser` | CLI 交互式命令解析（22条命令） |
| `FsCliApp` | 主入口 |

### 支持的命令

```
系统: format, save, info, exit
目录: mkdir, rmdir, ls, cd, pwd, tree
文件: create, open, close, read, write, delete
附加: rename, copy, move, stat
```
