## 前端交互（Vue）

| 交互方式 | 说明 |
|---------|------|
| **左键单击文件** | 在预览面板中打开文件内容 |
| **左键单击目录** | 进入该目录 |
| **右键文件/目录** | 弹出上下文菜单（打开/重命名/复制/移动/属性/删除） |
| **右键空白区域** | 弹出全局菜单（新建目录/新建文件/刷新/保存/信息） |
| **目录树左键** | 展开/收起子目录节点 |
| **目录树右键** | 弹出目录操作菜单 |
| **面包屑导航** | 点击路径段跳转 |
| **底部控制台** | 输入 CLI 命令 + `help` 查看完整命令列表 |

## 启动方式

### CLI 模式
```bash
cd filesystem-core
java -cp build/classes com.fs.FsCliApp
```

### REST API 模式
```bash
cd filesystem-core
java -cp build/classes com.fs.FsApiApp --port 8080
```

### 前端模式
```bash
cd filesystem-vue
npm run dev     # 开发模式
npm run build   # 构建
```
