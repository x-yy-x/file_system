## 启动方式

### CLI 模式
```bash
cd filesystem-core
java -cp build/classes com.fs.FsCliApp
```

### REST API 模式
```bash
cd filesystem-core
java -cp build/classes com.fs.FsApiApp --port 8080
```

### 前端模式
```bash
cd filesystem-vue
npm run dev     # 开发模式
npm run build   # 构建
```
